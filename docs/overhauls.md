# Overhauls

---

:::danger Spicy Mods Ahead
The mods in this section may change the game dramatically, and some are **very challenging**. Don't overdo it and remember that everything is **optional**!
:::

### [Charisma Tweaks](https://www.nexusmods.com/newvegas/mods/71310)

Barter and speech checks (And optionally, ALL skill checks) take into account both your charisma and the intelligence of the person you're speaking to. Optionally, it can use the charisma of the person you're speaking to instead.

### [Fatigue Overhaul - Non-Lethal and Pacifism Redone](https://www.nexusmods.com/newvegas/mods/78111)

This mod makes Fatigue *not* a completely useless mechanic. It also makes a pacifist playthrough actually feasible, if you're interested in that sort of thing.

### [Transporters - Immersive Fast Travel](https://www.nexusmods.com/newvegas/mods/77645)

This mod implements an alternative to fast-travel, adds a new faction, a new settlement, a quest, and... Moleholes? Check it out if you don't like vanilla fast travel.

:::note
We highly recommend that you **disable** fast travel when using this mod.
:::
